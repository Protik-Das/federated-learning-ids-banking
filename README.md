# 🔐 Secure Banks, Smarter Networks
## Federated Learning-Based IDS for Multi-Branch Banking Systems

📄 BSc Thesis | Department of CSE | University of Global Village | March 2026

---

## 📌 Abstract
A privacy-preserving Intrusion Detection System (IDS) for distributed 
banking environments using Federated Learning. Each branch trains locally — 
no raw data is shared — achieving **87% balanced accuracy** and **ROC-AUC: 0.89**.

---

## 🎯 Key Features
- ✅ Federated Learning with FedAvg algorithm
- ✅ Deep Neural Network (256→128→64→2 architecture)
- ✅ SMOTE-based class imbalance handling
- ✅ Complete data privacy — no raw data transfer
- ✅ Evaluated on CICIDS2018 dataset

## 🛠️ Technologies Used
- Python
- Federated Learning (FedAvg)
- Deep Neural Network
- SMOTE Resampling
- CICIDS2018 Dataset

## Results
| Metric | Score |
|--------|-------|
| Balanced Accuracy | 87% |
| ROC-AUC | 0.89 |
